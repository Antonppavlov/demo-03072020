# DEMO проект

Запуск тестов и сборка отчета

```mvn
clean test allure:serve
```

Пояснение к командам:
=========================

```mvn
clean - очистка проекта
```

```mvn
test - запуск тестов
```

```mvn
allure:serve - запуск allure отчетов
```
